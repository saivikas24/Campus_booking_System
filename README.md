# Campus Resource Booking System

A web app to manage booking of campus rooms and facilities with admin approval and conflict-free scheduling.

## Main Features

- Student registration & login
- Role-based access (Student/Admin)
- View available rooms and facilities
- Book resources with date & time slots
- Conflict-free booking (no double booking)
- Booking allowed only during college hours
- Maximum 2-hour booking limit
- Purpose required for classroom bookings
- Admin approval & rejection system
- Rejection reason sent to student
- Email notifications for booking updates
- Booking status tracking (Pending, Approved, Rejected, Completed)
- Admin dashboard with booking analytics
- Booking filters for admin
- Automatic completion of past bookings

->Tech Stack
PHP, MySQL, HTML, CSS, XAMPP, PHPMailer

->Installation

1.Clone or download the repository.
2.Move the project folder into XAMPP htdocs directory.
3.Start Apache and MySQL in XAMPP.
4.Create database campus_booking_system in Mysql.
5.Import the provided SQL file.
5.Update database credentials in config/db.php.
